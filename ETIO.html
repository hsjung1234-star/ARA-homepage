01

현장의 고통

타이밍 1ms의 오차가 수억 원의 손실을 만듭니다

3가지 핵심 문제
1
보이지 않는 손실

카메라도 있고 조명도 있는데, 촬영 타이밍이 1ms만 어긋나도 정상 제품이 불량으로 판정되어 버려집니다. 연간 수억 원 규모의 양품 폐기 손실이 발생합니다.

2
반복되는 재검사

원인 모를 '과검' 때문에 인건비가 낭비되고, 정작 중요한 불량은 놓치는 위험에 노출됩니다. 재검사 사이클이 라인 전체 속도를 저하시킵니다.

3
이중 장비 구성의 복잡성

기존에는 트리거 컨트롤러와 IO카드를 별도로 구축해야 했습니다. 장비가 늘수록 배선이 복잡해지고, 고장 포인트와 유지보수 비용도 함께 증가합니다.

03

ETIO의 해결책

트리거 분배기 + IO카드, 단 하나의 장비로 통합

트리거 + IO 통합 단일 장비
Core Technology
FPGA 기반 하드웨어 직접 신호 처리

소프트웨어 스케줄링에 의존하는 PC·PLC 방식과 달리, ETIO는 FPGA가 직접 신호를 처리합니다. OS 오버헤드가 전혀 없어 지터 100ns 미만, 레이턴시 1µs 미만의 결정적 타이밍을 보장합니다. 사람이 눈을 깜빡이는 시간의 100만 분의 1 속도로 반응합니다.

Patent
Working Area 기능 (특허)

센서 또는 PLC 신호와 연동하여 제품이 실제로 있는 구간에서만 트리거를 발생시킵니다. 불필요한 촬영과 데이터 처리를 원천 차단하여 오트리거를 제거합니다.

2-in-1
트리거 분배기 + IO카드 통합

1개 엔코더 신호를 최대 8채널로 동시 분배하면서, PLC 디지털 I/O 인터페이스 역할까지 수행합니다. 별도 IO보드 불필요 — 단일 장비로 완성됩니다.

Noise Immunity
강력한 노이즈 차단 설계

내장 Digital Noise Filter와 입출력 전 구간 Galvanic Isolation(광절연) 설계로 현장 노이즈 환경에서도 안정적으로 동작합니다.

Non-Volatile
EEPROM 비휘발성 메모리

설정값을 내부 EEPROM에 저장합니다. 전원 재투입 시 별도 설정 없이 즉시 가동되어 라인 재가동 시간을 최소화합니다.

Compensation
Backlash · Noise 보정

Compensational Mode로 모터 Backlash 현상과 Encoder Noise를 하드웨어 레벨에서 보완합니다. Working Direction으로 왕복운동 검사도 지원합니다.

Working Area 특허 기능

제품이 있는 구간에서만 트리거 — 오검출 원천 차단

특허 등록 완료

바깥 영역 클릭 또는 ✕ 버튼으로 닫기

Working Area는 에이알에이 일렉트로닉스이 특허를 보유한 핵심 기술로, 센서 또는 PLC 신호를 기준으로 제품이 실제로 존재하는 구간에서만 엔코더 트리거를 발생시킵니다. 불필요한 트리거를 원천 차단하여 데이터 처리 부하와 과검출을 제거합니다.

Working Direction 기능과 결합하면 왕복운동 검사 장치에서 특정 방향 이동 시에만 트리거가 발생하여 소프트웨어 개발 부담을 크게 줄입니다.

Virtual Encoder 모드에서도 동작
실제 엔코더 없이도 Working Area 기능을 사용할 수 있어, Signal Generator로서 다양한 테스트 환경에 활용 가능합니다.
